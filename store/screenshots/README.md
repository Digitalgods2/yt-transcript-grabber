# Screenshots to capture

The Chrome Web Store requires **at least one** screenshot. Recommended size:
**1280×800** (or 640×400). Provide at least three for a professional listing.

Capture these on a real YouTube watch page with the extension installed and
the toolbar icon pinned:

1. **popup-closed.png** — the whole Chrome window on a YouTube video, with the
   toolbar icon visibly pinned and the transcript panel open on the right.
2. **popup-open.png** — the popup open, "Include timestamps" ticked, showing
   the filled preview textarea after a successful grab. Status line shows
   "Copied N chars to clipboard."
3. **downloaded.png** — the Downloads bar / Downloads folder showing the
   saved `<video title>.txt` file opened in a text editor with the first few
   transcript lines visible.

Save the PNGs into this folder, then drag them into the Dev Console's
screenshot uploader when creating the listing.
